==============
For Developers
==============

Zielgruppe: **Developer**

Mitwirkung
==========

Mitwirkungen sind für den Erfolg von Open-Source-Projekten von entscheidender Bedeutung, beschränken sich jedoch nicht nur auf Code-Beiträge. Es kann noch viel mehr getan werden:

* Dokumentantion verbessern
* Fragen in diversen Foren wie stackoverflow.com oder typo3forum.de beantworten

Workflow
^^^^^^^^

Bevor mit einer Code-Änderung begonnen wird, bitte immer ein Issue unter https://github.com/digedag/cfc_league/issues erstellen. Dies ist hilfreich , wenn man sich nicht sicher ist, ob eine Änderung akzeptiert wird und erleichtert später auch das Mergen eines Pull Requests, wenn man den Kontext nochmals nachlesen lann.

Letzte Version vom Git-Repository
"""""""""""""""""""""""""""""""""

Forke das Repository https://github.com/digedag/cfc_league und erstelle einen Pull Request mit den Änderungen.
